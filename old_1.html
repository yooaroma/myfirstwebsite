HTML 이란 무엇인가

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applicateions. web browsers receive HTML documents from a web server or from local storage and render them into mutimedia web pages. HTML descrives the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements ar the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements ar delineated by tags, written using angle brackets.
